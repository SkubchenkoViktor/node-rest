# node-rest-mysql

This is a boilerplate for using Node.js with Express to host a RESTful API for a MySQL/MariaDB Database.

## Installation

1. `npm install`

2. create a `credentials.j`s file in the root of the repo

   ```javascript
   module.exports = {
   	host     : 'localhost',
   	user     : 'me',
   	password : 'secret',
   	database : 'my_db'
   }
   ```
